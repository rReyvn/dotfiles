- Dotfiles Installation
  - Clone Repository
    ```sh
    git clone git@github.com-reyvn:rReyvn/dotfiles.git && cd dotfiles
    ```
  - Install
    ```makefile
    make rice
    ```
  - Install terminal dots only
    ```makefile
    make rice-term
    ```
  - Uninstall
    ```makefile
    make unrice
    ```
